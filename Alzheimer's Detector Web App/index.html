Alzheimer's Disease Detection

This interactive web application demonstrates a machine learning model designed to assist in the early detection of Alzheimer's Disease. It uses a simple, explainable model to not only predict the outcome but also to show which factors most influence the prediction.

How it works:

  • **Data Input:** We collect key patient data points.
  • **Model Logic:** A simplified, explainable model processes the data.
  • **Prediction & Explainability:** The model predicts a result and explains the primary factors behind the decision.
